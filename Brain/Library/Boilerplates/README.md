# Boilerplates

Copy-paste snippets Claude (and you) want to reuse. Keep them **minimal** — a boilerplate should be the smallest-thing-that-works, not a full feature.

## Structure

One file per snippet: `<language>-<slug>.md` (e.g. `ts-ipc-handler.md`).

```markdown
# <Snippet name>

## Use case

<One line>

## Snippet

\`\`\`<lang>
<code>
\`\`\`

## Notes

<Optional: caveats, variants>
```
