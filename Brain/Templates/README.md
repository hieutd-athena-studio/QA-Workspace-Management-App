# Templates

Copy-paste starters. Keep templates in sync with the conventions referenced from [[Architecture]].

## Available

- [[ADR-template]] — Architecture Decision Record
- [[Journal-template]] — End-of-session journal entry
- [[Context-Map-template]] — Project Home Base (bootstrapping a new project)
- [[Lesson-Learned-template]] — Post-mortem for a bug that took > 30 min

## Convention

- Templates are **valid markdown** — they should render cleanly if opened directly.
- Required fields are marked `<REQUIRED>`; optional `<optional>`.
- Replace any `<PLACEHOLDER>` text; don't leave empty angle-bracket tokens.
